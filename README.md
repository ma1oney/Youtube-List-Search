# YouTube List Search 🔎

YouTubeのプレイリスト閲覧と管理を効率化するためのChrome拡張機能です。何千もの動画を含む長いプレイリストでも、検索機能で目的の動画に瞬時にアクセスできます

---

## ✨ 機能 (Features)

### 1. 🔍 高速リアルタイム検索 (フィルタリング)
リストの上部に埋め込まれたシンプルな検索バーを使用して、ロードされた全動画データに対して検索を実行できます。

* キーワードを入力するたびに、タイトルに一致する動画だけが瞬時にフィルタリングされ表示されます。
* 膨大なリストから目的の動画を探す手間を大幅に削減します。

### 2. 🖥️ UIの最適化
YouTubeのリストページに溶け込むシンプルなUIで、既存のデザインを崩さずに必要な機能だけを提供します。

---

## 🛠️ インストール方法 (Installation)

この拡張機能はまだChromeウェブストアでは公開されていません。以下の手順で**デベロッパーモード**を使用してインストールしてください。

### 1. 📂 ファイルのダウンロード
このリポジトリ全体をZIPファイルとしてダウンロードし、任意の場所に解凍します。（または、Gitでクローンします）
